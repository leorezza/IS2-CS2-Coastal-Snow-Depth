atl03-cs2-snow-depth

This repository contains MATLAB scripts developed for the processing of
ICESat-2 ATL03 photon data and CryoSat-2 observations in the coastal
Arctic with the purpose of snow depth estimation over sea ice.

The code was produced as part of a Master’s thesis carried out at the
Technical University of Denmark (DTU), in collaboration with UNIS (The
University Centre in Svalbard), and is intended for research and 
reproducibility purposes. Input data are not provided and must be 

obtained from the original providers.
